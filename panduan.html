Panduan Penggunaan

Panduan singkat untuk menggunakan My Storage Scanner v2.0

1 Pilih Mode Scan

Setelah membuka aplikasi, pilih salah satu mode:

  • Scan Cepat: Untuk scan drive C: saja (rekomendasi pemula)
  • Scan Kustom: Pilih drive dan folder sendiri
  • Pencari Duplikat: Khusus mencari file identik
2 Download & Jalankan Scanner

Download scanner.exe dan jalankan di komputer Anda:

  • Klik tombol "Download Scanner.exe ( Exe dijamin Aman dan bebas dari Virus)"
  • Simpan file di folder yang mudah diakses
  • Double-click scanner.exe untuk menjalankan
  • Tunggu proses scan selesai (Scanned Succesfully akan muncul di tampilan exe nya)
  • Maka akan muncul file Json baru bernama scan_result.Json yang berisi hasil scannya.
Pastikan scan_config.json berada di folder yang sama dengan scanner.exe
3 Analisis & Pembersihan

Upload hasil scan ( scan_result.json) dan lakukan pembersihan:

  • Upload file scan_result.json ke website
  • Review file yang ditemukan scanner
  • Centang file yang ingin dihapus
  • Download dan jalankan cleaner.exe
PERINGATAN: Selalu backup data penting sebelum menghapus file
