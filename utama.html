DETAIL SETIAP MODE

A. MODE CEPAT (Quick Scan)

Untuk pemula yang ingin scan drive C: saja

  1. Klik "Scan Cepat (Drive C:)"
  2. Langsung ke Langkah 2 di atas

B. MODE KUSTOM (Pilih Drive Sendiri)

Untuk scan beberapa drive dengan konfigurasi khusus

  1. Klik "Scan Kustom (Multi-Drive)"
  2. Pilih drives yang ingin discan
  3. (Opsional) Tambah folder custom
  4. Atur pengaturan scan
  5. Klik "Generate & Download Config"
  6. Lanjut ke Langkah 2

C. MODE CARI DUPLIKAT

Khusus mencari file duplikat

  1. Klik "Pencari File Duplikat"
  2. Pilih drives (minimal 1)
  3. Pilih jenis file: gambar, video, dokumen, dll
  4. Atur ukuran minimum: 100 KB
  5. Klik "Mulai Scan Duplikat"
  6. Lanjut ke Langkah 2

TIPS & PERINGATAN

PERINGATAN PENTING

  • Jangan hapus file system (windows, program files)
  • Backup data penting sebelum menghapus
  • Review dulu sebelum klik delete
  • Cleaner hanya hapus file yang dipilih di website

TIPS CEPAT

  • File besar (>100MB) ‚Üí Centang untuk hapus jika tidak penting
  • File duplikat ‚Üí Gunakan "Smart Select" untuk otomatis pilih
  • File .tmp & .log ‚Üí Aman untuk dihapus (temporary files)
  • Drive penuh ‚Üí Scan semua drive untuk cari file besar

Apa Itu ds/dt?

ds/dt adalah turunan pertama dari fungsi penyimpanan terhadap waktu. Dalam konteks program ini:

s (Storage)

Total data yang tersimpan di komputer (bytes, MB, GB)

Variabel dependen yang diukur

t (Time)

Waktu pengukuran (detik, menit, hari)

Variabel independen

ds/dt

Laju perubahan storage per satuan waktu

Turunan pertama s terhadap t

Rumus Matematis:

ds/dt = limŒît‚Üí0 [s(t+Œît) - s(t)] / Œît

Dalam praktik: Œîs/Œît antara dua scan

Interpretasi Hasil ds/dt

ds/dt > 0

Penyimpanan Bertambah

Ada data baru masuk

ds/dt < 0

Penyimpanan Berkurang

Data berhasil dibersihkan

ds/dt = 0

Stabil

Tidak ada perubahan signifikan

|ds/dt| besar

Perubahan Cepat

Perlu investigasi

Contoh Kasus Nyata:

Jika ds/dt = +5 MB/detik:

  • Storage bertambah 300 MB/menit
  • 18 GB/jam
  • 432 GB/hari

Tindakan yang diperlukan:

  • Cek program yang sedang berjalan
  • Monitor download/upload
  • Cek log aplikasi
